PETA Log 04-26-2012

Work done:

1. Read length modifed to 100bp. Results based on simulated dataset: Simulation Result

2. To support not strand specific mode in simulation,  assembly and evaluation. But the result needs to be tuned. Simulation Result.

  • Simulation: when producing a read, if the parameter is set, with a probability of 50% it will be from the reverse complementary strand;
  • Hashing: remains the same;
  • Assembly: for single extension, when aligning the query to the libraries, align the reverse complement of the query as well, add it to the current pool as well; for extension by mates, check the reverse complement of the query as well;
  • Evaluation: include all hits instead of only the hits from the sense strand.
