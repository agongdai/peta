PETA Log 19-04-2012

1. Improvement: PETA starts extension from some random read which is not used yet. Modify to prune the graph every time after a random read extension is finished. That would avoid complicated graph in the end.

2. Real dataset candidates for evaluation:

1) Schizosaccharomyces pombe: strand-specific 68PE Illumina reads, 9.4G bases, 9GB; http://www.ncbi.nlm.nih.gov/sra/SRX040571; Used by Trinity and this paper: http://www.biomedcentral.com/1471-2105/12/S14/S2#B3

Evaluation (Adopted by Oases and Trinity): Get the reference genome: http://www.pombase.org/downloads/datasets; get asssembly result by Cufflinks/Scripture, treate them as target transcripts. Then compare with PETA results;

2) Human/mouse RNA-seq dataset. (dataset used by Oases is not strand-specific).

3. Plot length distribution for target graph and PETA graph
